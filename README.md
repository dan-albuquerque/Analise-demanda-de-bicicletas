# 📈 Previsão da Demanda de Bicicletas em Londres

Este projeto utiliza dados de séries temporais para prever a demanda por bicicletas compartilhadas em Londres, com foco na análise de padrões sazonais e variáveis ambientais.

# 🧠 Objetivo
Analisar dados históricos de aluguel de bicicletas para:

Entender padrões de comportamento do usuário.

Investigar variáveis sazonais que impactam a demanda.

Aplicar testes estatísticos e modelos preditivos.

Apoiar a tomada de decisão estratégica para empresas do setor.

# 🔍 Análises Realizadas
Exploração de Dados: análise descritiva de variáveis numéricas e categóricas.

Testes Estatísticos: Mann-Whitney para comparação sazonal (outono x primavera).

Visualizações: gráficos de distribuição, boxplots e séries temporais.

Modelagem Preditiva (caso presente): modelos como ARIMA, Prophet ou LSTM para previsão de demanda.

# 📊 Dataset
O conjunto de dados foi obtido de London Bike Sharing Dataset via Kaggle e contém:

Contagem horária de bicicletas alugadas.

Variáveis meteorológicas (temperatura, umidade, vento).

Estações do ano e feriados.

# 📌 Algumas Conclusões
A demanda por bicicletas varia significativamente entre as estações.

Estratégias diferenciadas por sazonalidade podem otimizar a operação.

Previsões de demanda auxiliam no planejamento de frota e pontos de retirada.
